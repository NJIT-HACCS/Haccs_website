# Hispanic Association of Computing College Students (HACCS) - Project Repository

This repository contains the frontend and backend code for the [Project Name] application. Our application uses the MERN stack: MongoDB, Express.js, React, and Node.js.

## Getting Started

To set up and run this project locally, follow these steps:

### Prerequisites

Ensure you have the following installed:

* [Node.js](https://nodejs.org/)
* [MongoDB](https://www.mongodb.com/try/download/community)

### Clone the Repository

```bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```

## Install Dependencies

Navigate to both the backend and frontend directories to install the required packages.

### Backend:

```bash
cd ./backend
npm install
```
### Frontend

```bash
    cd ./frontend
    npm install
```

### Set Up Environment Variables
Currently we dont have any service running however you can create an empty .env file both in frontend and backend

``` bash
    cd ./frontend
    touch .env
    cd ./backend
    touch .env
```

### Run the Application 
```bash
    cd ./frontend
    npm start
```
The frontend should be available at http://localhost:3000 and the backend at http://localhost:5000 (or the specified port).

## Contributing

We welcome contributions from everyone! To maintain quality and consistency, please follow this workflow:

### Pull Request Workflow

1. **Fork the repository** and clone it to your local machine.
2. **Create a new branch** for your changes:
    ```bash
    git checkout -b feature/your-feature-name
    ```
3. **Make your changes** and commit them with a meaningful message.
4. **Push your branch** to your forked repository:
    ```bash
    git push origin feature/your-feature-name
    ```
5. **Submit a pull request**:
   - Describe the changes you made.
   - Link any relevant issues.
   - Assign at least one reviewer from the team.
6. **Code Review**: A reviewer will review your pull request. Please address any comments or feedback provided.

Thank you for contributing!
